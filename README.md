## Marketing & Sales Performance Analysis (Python Project)

A complete end-to-end marketing and sales analytics project using Python, focused on understanding revenue drivers, campaign effectiveness, platform performance, and regional sales behavior.
This project demonstrates real-world analytical skills used in data-driven marketing and business intelligence.

### Project Overview

This project analyzes a multi-dimensional marketing dataset containing revenue, ad spend, clicks, item sales, regional data, client information, and platform performance.
The goal is to uncover insights that help businesses optimize budget allocation, identify high-value customers, and improve overall marketing ROI.

### Key Insights

Revenue is heavily concentrated in a few key regions and top clients.
Some platforms deliver high ROI, while others show inefficiencies.
Several campaigns can be scaled, while underperformers need rework.
Seasonal patterns exist, making the data suitable for forecasting.
Strong correlation between Clicks, ItemSales, and Revenue indicates predictable performance drivers.
